# 🎲 SUNLON - Hệ Thống Quản Lý Key & Dự Đoán Tài Xỉu

[![Python](https://img.shields.io/badge/Python-3.11-blue.svg)](https://www.python.org/)
[![Telegram](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://t.me/)

## 📌 Giới Thiệu

SunLon là hệ thống quản lý key và dự đoán Tài Xỉu chuyên nghiệp, bao gồm:

- 🤖 **Telegram Bot** - Quản lý key, tạo key, thống kê, xuất database
- 💻 **Desktop Client** - Ứng dụng GUI hiển thị dự đoán với xác thực key
- 🌐 **Web Server** - Health check và API để đồng bộ database

## 🚀 Tính Năng

### Telegram Bot
- ✅ Tạo key với thời hạn tùy chỉnh
- ✅ Quản lý danh sách key
- ✅ Kiểm tra key cho người dùng
- ✅ Thống kê số lượng key
- ✅ Vô hiệu hóa/xóa key
- ✅ Tải database keys
- ✅ Web server cho health check

### Desktop Client
- 🎲 Hiển thị dự đoán Tài Xỉu realtime
- 📊 Thống kê tỷ lệ chính xác
- 📜 Lịch sử dự đoán
- 🌓 Dark/Light mode
- 🔐 Xác thực key qua database
- 🔄 Tự động đồng bộ database từ server
