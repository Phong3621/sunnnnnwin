# 🎲 SunLon - Hệ thống quản lý key và dự đoán Tài Xỉu

## 📌 Giới thiệu
SunLon là hệ thống quản lý key và dự đoán Tài Xỉu chuyên nghiệp, bao gồm:
- **Telegram Bot**: Quản lý key, tạo key, thống kê
- **Client**: Ứng dụng desktop hiển thị dự đoán
- **Server API**: Xác thực key từ xa

## 🚀 Tính năng
### Telegram Bot
- Tạo key với thời hạn tùy chỉnh
- Quản lý danh sách key
- Thống kê số lượng key
- Kiểm tra key cho người dùng
- Vô hiệu hóa/xóa key

### Client
- Hiển thị dự đoán Tài Xỉu realtime
- Thống kê tỷ lệ chính xác
- Lịch sử dự đoán
- Dark/Light mode
- Xác thực key qua Telegram
